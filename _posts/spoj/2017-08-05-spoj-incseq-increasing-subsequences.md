## Đề bài :
Cho 1 dãy N phần tử (1 <= N <= 10^4). Đếm số dãy con tăng độ dài k (1 <= k <= 50 && k <= N).

# Input :
- Dòng đầu gồm 2 số n, k
- N dòng tiếp theo gồm số của dãy.

# Output :
- Gồm 1 dòng là kết quả của bài toán.

# Example :
```
Input:
4 3
1
2
2
10

Output:
2
```
